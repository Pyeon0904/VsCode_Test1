전역 속성

모든 HTML 요소에서 사용할 수 있는 속성

id 속성

요소에 고유한 식별 ID(이름)를 지정
문서 내 다른 요소와 중복될 수 없음
이름은 하나 이상의 문자가 포함되어 있어야 하며, 공백문자가 없어야함
HTML, CSS, Javascript 에서 요소를 구분하기 위한 용도로 주로 사용됨

class 속성

요소에 하나 이상의 이름을 지정
문서 내 여러 요소에 같은 이름 지정 가능
이름은 알파벳, 숫자, 하이픈, 밑줄로 구성(시작은 알파벳으로 지정해야 함)
HTML, CSS, Javascript 에서 요소를 구분하기 위한 용도로 주로 사용됨

hidden 속성

요소가 아직 또는 더 이상 관련이 없음을 지정(요소 숨김 지정)
다른 조건이 충족 될 때 까지 사용자가 요소를 보지 못하게 하는데 사용
Javascript 를 이용해 속성 변경 가능

title 속성

요소에 대한 추가 정보를 지정
요소 위로 마우스를 이동하면 추가 정보가 툴팁 텍스트로 표시됨

style 속성

요소에 대한 인라인 CSS 스타일 지정
여러 스타일을 지정할 경우 세미콜론(;)으로 구분
style 태그 또는 외부 스타일시트에 지정된 스타일보다 우선 적용됨
