멀티미디어 관련 태그

이미지 태그(img)

웹 페이지에 사진이나 그림을 삽입할 때 사용하는 태그
이미지는 기술적으로 HTML 페이지에 삽입 되지 않음
링크를 이용해 이미지파일을 별도로 가져와 표시해줌

이미지 삽입1 : 다른 사이트 이미지 주소 복사

이미지 삽입2 : 이미지 파일을 서버에 저장

서버의 특정 경로에 이미지 파일 저장 : resources/images/kh_logo.jpg

이미지 삽입3 : 이미지 설명 텍스트 지정

이미지가 제대로 표시되지 않는 경우에만 텍스트 표시됨

이미지 삽입4 : 이미지 크기 지정

map 태그

이미지를 구역별로 나누어 링크를 걸어주는 태그
영역을 지정하는 별도의 map을 만들어 놓고 img 태그에서 해당 map을 지정(usemap속성)해서 사용

이미지에 링크 지정

오디오 태그(audio)

웹 브라우저에서 플러그인의 도움없이 음악을 재생할 수 있게 만들어주는 태그

비디오 태그(video)

웹 브라우저에서 플러그인의 도움없이 미디어를 재생할 수 있게 만들어주는 태그

동영상 재생 : 컨트롤 표시(controls)

동영상 재생 : 재생 전 출력할 이미지 지정(poster), 크기지정(width/height)
