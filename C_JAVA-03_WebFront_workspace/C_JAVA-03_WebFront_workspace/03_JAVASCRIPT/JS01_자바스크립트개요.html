자바스크립트 개요

웹 브라우저에서 많이 사용하는 인터프리트 방식의 객체 지향 프로그래밍 언어
브라우저에서 동적인 기능을 수행할 수 있게 해줌

자바스크립트 선언

HTML의 script 태그를 이용해서 javascript로 해석할 영역을 만들어 줌
type="text/javascript" 는 기본값이기 때문에 생략 가능

자바스크립트 위치

페이지 내 어느 영역에서나 사용 가능하지만 head나 body내부에서 사용하는 것을 권장

자바스크립트 작성 방식 1 : internal(내부) 방식

가장 일반적인 방식으로 HTML 파일 내 script 태그를 사용하여 코드를 작성하는 방식

자바스크립트 작성 방식 2 : inline(인라인) 방식

HTML 태그의 이벤트 핸들러 속성을 이용해 자바스크립트를 실행시키는 방식

코드 직접 실행

javascript: 은 생략 가능

다른방식으로 미리 만들어놓은 코드(함수) 실행

자바스크립트 작성 방식 3 : external(외부) 방식

자바스크립트 코드 부분을 외부 파일로 저장해서 작성
파일의 확장자는 .js
script 태그의 src 속성으로 파일을 불러와서 실행

자바스크립트 지원 안하는 경우

HTML의 noscript 태그를 이용해 지원하지 않을 경우 출력 문구 지정 가능
