내장 함수

escape(), unescape()

알파벳 대/소문자, 0~9 숫자, @*-_+./ 를 제외한 모든 문자열을 인코딩


encodeURI(), decodeURI()

인터넷 주소에 사용되는 문자열 인코딩 - : ; / = ? & 는 변환 안함


eval()

문자열을 자바스크립트 코드로 변환해서 실행하는 함수

isFinite(), isNaN()

isFinite() : number 형 데이터가 맞는지 확인(맞으면 true, 아니면 false)
IsNaN() : number형 데이터가 아닌지 확인(아니면 true, 맞으면 false)
