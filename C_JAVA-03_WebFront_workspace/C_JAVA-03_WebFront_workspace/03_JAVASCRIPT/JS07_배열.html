배열 초기화

배열 메소드

indexOf() : 배열에서 요소가 위치한 인덱스 리턴

concat() : 두 개 또는 세개의 배열을 결합

join() : 배열을 결합해서 문자열로 반환

reverse() : 배열의 순서를 뒤집어 주는 메소드

sort() : 배열을 내림차순 혹은 오름차순으로 정렬

push() : 배열의 맨 뒤에 요소 추가, pop() : 배열의 맨 뒤에 요소 제거

shift() : 배열의 맨 앞에 요소 제거, unshift() : 배열의 맨 앞에 요소 추가

slice() : 배열의 요소 선택 잘라내기, splice() : 배열의 index위치의 요소 제거 및 추가

toString() : 배열을 문자열로 반환
