JQuery 개요

경량 Javascript 라이브러리
복잡한 Javascript 코드를 손쉽게 구현하기 위해 개발됨

JQuery 장점

1. DOM과 관련된 처리를 쉽게 구현할 수 있음
2. ajax통신, 이벤트 처리 등을 폭넓게 지원함
3. 별도의 플러그인을 통해 차트, 슬라이드쇼, 테이블 등을 간단히 구현 가능

JQuery 연결

1. head 요소안에 script 태그를 이용해서 JQuery 파일을 포함시켜줌
- JQuery 홈페이지에서 .js 파일을 다운받아서 연결하는 방법
- CDN(Content Delivery Network)를 통해 연결하는 방법
2. script 태그 내에서 JQuery 문법으로 코드 작성

#(document).ready()

페이지를 모두 로드한 후 ready 메소드를 실행
자바스크립트의 document.onload 와 같은 개념의 메소드
