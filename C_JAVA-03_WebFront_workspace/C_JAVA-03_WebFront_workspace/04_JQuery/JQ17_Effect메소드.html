Effect 메소드

페이지에 에니메이션 효과를 만들기 위한 메소드 집합

animate() 메소드

현재의 css속성을 설정한 값으로 차츰 변경되는 효과를 주는 메소드

show(), hide(), toggle() 메소드

문서 객체를 크게 하여 보여주거나 작게하여 사라지게 함


fadeIn(), fadeOut(), fadeToggle() 메소드

문서 객체를 진하게 하여 보여주거나, 희미하게 하여 사라지게 함


slideDown(), slideUp(), slideToggle() 메소드

선택된 요소 위치에서 내려가 요소를 보여주거나 밀고 올라가 숨겨주는 효과
