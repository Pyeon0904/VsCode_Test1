CSS(Cascading Style Sheet)

HTML 문서의 스타일을 설명하는 언어
HTML 컨텐츠가 화면에 어떻게 표시 될 것인지를 나타내는 규약

CSS 사용 방법 1

인라인 스타일 : 각 요소의 style 속성으로 지정

영역

CSS 사용 방법 2

내부 스타일 시트 : style 태그로 지정

영역2

CSS 사용 방법 3

외부 스타일 시트 : 별도의 .css 파일을 생성해서 지정
head 에 link 태그를 이용해서 파일을 불러와야 적용됨

영역3

CSS 적용 우선순위

1. 인라인 스타일
2. 내부스타일시트 == 외부스타일시트(마지막으로 읽은 스타일 적용)
3. 브라우저 기본값

CSS Syntax(형식)

Selector { Property : Value; [...] }

Selector(선택자) : 스타일을 적용할 HTML 요소를 지정
Property(속성) : 적용할 CSS 속성
Value(값) : 속성에 적용할 값
