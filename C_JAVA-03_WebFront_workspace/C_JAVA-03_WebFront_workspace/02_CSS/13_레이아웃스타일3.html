콘텐츠영역

padding

테두리와 컨텐츠 영역 사이의 거리를 조절하는 속성
padding : 전체;
padding : 상하 좌우;
padding : 상 좌우 하;
padding : 상 우 하 좌;

상하좌우 전체 padding: 100px

콘텐츠영역

위쪽: 50px, 왼쪽: 100px

콘텐츠영역

상하: 50px, 좌우: 30px

콘텐츠영역

하: 50px, 좌우: 10px, 상: 0px

콘텐츠영역

margin

요소들간의 간격을 조절하는 속성
사용법은 padding과 동일

상하좌우 전체 margin: 50px

콘텐츠영역

하: 50px, 좌우: 10px, 상: 0px
