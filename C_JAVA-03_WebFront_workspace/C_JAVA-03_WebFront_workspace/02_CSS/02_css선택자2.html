 후손선택자 : 선택자  후손선택자 { 설정내용; }

동위선택자

동위 관계에서 뒤에 위치한 태그를 선택

바로 뒤의 요소 선택 : 선택자a + 선택자b { 설정내용; }
뒤에 있는 모든 요소 선택 : 선택자a ~ 선택자b { 설정내용; }

반응선택자

사용자의 움직임에 따라 달라지는 선택자

사용자가 클릭한 태그 - 선택자:active { 설정내용; }
마우스가 올라가있는 태그 - 선택자:hover { 설정내용; }

상태선택자

입력양식의 상태에 따라 선택되는 선택자

체크 상태의 input 태그 선택 - input태그:checked { 설정내용; }
사과 바나나 오렌지 
초점이 맞춰진 input 태그 선택 - input태그:focus { 설정내용;}

비밀번호 : 
사용가능한 input태그 선택 - input태그:enabled { 설정내용;} 
사용불가능한 input태그 선택 - input태그:disabled { 설정내용;}
